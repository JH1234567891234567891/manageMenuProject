# Menu Project

---
간단한 메뉴 관리 프로그램
- 관리되는 메뉴는 **메뉴 번호**, **메뉴명**, **가격**, **카테고리**, **판매여부**  
- 메뉴 데이터 관리 기능은 **메뉴 추가**, **삭제**, **수정**, **전체 출력**
## 프로젝트 구조
```
controller
  |
  ├ Controller
  ├ HandlerMapping
  | ...
  |
main
  ├ menuDataMain
service
  ├ menuDataService
exception
  ├ menuDataException
 vo
  └ menuDataVO
```
